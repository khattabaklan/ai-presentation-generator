    The Workflow Behind This Tool

    This product was born from a real university workflow — here's the story.

    The Problem

    University students spend hours creating presentations for every assignment. The process is tedious: read the instructions, research the topic, organize the content into slides, design the layout, and write speaker notes. For a single 10-slide presentation, this can take 3-5 hours.

    The Original Workflow

    Khattab built a personal workflow using Claude AI that automated this entire process:

    1. Input: Paste the assignment instructions into Claude
    2. Content Generation: Claude analyzes the requirements and generates structured slide content — titles, bullet points, and detailed speaker notes for each slide
    3. Slide Creation: A Node.js script uses pptxgenjs to convert the structured content into a polished PowerPoint file
    4. Script Creation: Another script generates a Word document with the full speaker script, organized by slide

    The Result

    What used to take 3-5 hours now takes under 30 seconds. The presentations are well-structured, professionally formatted, and include everything needed to present confidently — including detailed speaker notes that read naturally.

    From Workflow to Product

    This tool packages that exact workflow into a simple web interface. Upload your assignment, pick a theme, and download your presentation. One free generation to try it out, then unlimited access with a Pro subscription.
